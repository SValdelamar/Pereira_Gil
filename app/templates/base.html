Nuestra Misión

El Centro Minero del SENA es una institución líder en la formación profesional integral en el sector minero y energético de Colombia. Nos dedicamos a formar talento humano competente y competitivo que contribuya al desarrollo sostenible del país.

Sistema de Gestión de Laboratorios

Este sistema ha sido desarrollado para optimizar la administración de recursos educativos:

  • Gestión eficiente de equipos e inventario
  • Control de acceso con reconocimiento facial
  • Reservas automatizadas de equipos y espacios
  • Trazabilidad completa de uso de recursos
  • Reportes en tiempo real para toma de decisiones
Tecnología e Innovación

Implementamos tecnologías de vanguardia como inteligencia artificial, reconocimiento facial y visual, para brindar una experiencia moderna y segura a nuestros aprendices e instructores.

Política de Privacidad

Última actualización: Octubre 2025

1. Información que Recopilamos

El Sistema de Gestión de Laboratorios del Centro Minero SENA recopila la siguiente información:

  • Datos de identificación: Nombre, documento de identidad, correo electrónico
  • Datos biométricos: Imágenes faciales para control de acceso
  • Datos de uso: Registros de acceso, reservas de equipos, historial de uso
  • Datos técnicos: Dirección IP, tipo de navegador, sistema operativo
2. Uso de la Información

Utilizamos la información recopilada para:

  • Gestionar el acceso a laboratorios y equipos
  • Verificar la identidad de usuarios mediante reconocimiento facial
  • Generar reportes de uso y estadísticas
  • Mejorar la seguridad y funcionamiento del sistema
  • Cumplir con requisitos legales y normativos
3. Protección de Datos

Implementamos medidas de seguridad técnicas y organizativas para proteger sus datos personales:

  • Cifrado de datos sensibles
  • Control de acceso basado en roles
  • Auditoría de accesos y cambios
  • Respaldos periódicos de información
4. Derechos del Usuario

Como usuario del sistema, usted tiene derecho a:

  • Acceder a sus datos personales
  • Solicitar corrección de datos inexactos
  • Solicitar eliminación de datos (sujeto a requisitos legales)
  • Oponerse al procesamiento de sus datos
5. Retención de Datos

Los datos personales se conservan durante el tiempo necesario para cumplir con los fines para los que fueron recopilados y de acuerdo con las obligaciones legales aplicables.

Para ejercer sus derechos o realizar consultas sobre privacidad, contacte a: gilcentrominero@gmail.com
Términos de Uso

Última actualización: Octubre 2025

1. Aceptación de Términos

Al acceder y utilizar el Sistema de Gestión de Laboratorios del Centro Minero SENA, usted acepta estar sujeto a estos términos y condiciones de uso.

2. Uso Autorizado

El sistema está destinado exclusivamente para:

  • Aprendices matriculados en programas del Centro Minero SENA
  • Instructores y personal autorizado del SENA
  • Administradores del sistema
3. Responsabilidades del Usuario

Como usuario del sistema, usted se compromete a:

  • Mantener la confidencialidad de sus credenciales de acceso
  • No compartir su cuenta con terceros
  • Utilizar el sistema únicamente para fines educativos y administrativos legítimos
  • No intentar acceder a áreas restringidas o datos de otros usuarios
  • Reportar cualquier problema de seguridad o uso indebido
4. Uso de Reconocimiento Facial

Al registrar su rostro en el sistema, usted autoriza:

  • El almacenamiento de sus datos biométricos faciales
  • El uso de reconocimiento facial para control de acceso
  • El procesamiento de imágenes para verificación de identidad
5. Reservas y Uso de Equipos
  • Las reservas deben realizarse con anticipación según disponibilidad
  • El usuario es responsable del cuidado de los equipos durante su uso
  • Cualquier daño debe reportarse inmediatamente
  • El incumplimiento puede resultar en suspensión de privilegios
6. Propiedad Intelectual

Todo el contenido, diseño y funcionalidad del sistema son propiedad del SENA y están protegidos por leyes de propiedad intelectual.

7. Modificaciones

El SENA se reserva el derecho de modificar estos términos en cualquier momento. Los cambios entrarán en vigor inmediatamente después de su publicación.

8. Suspensión de Servicio

El SENA se reserva el derecho de suspender o terminar el acceso de cualquier usuario que viole estos términos.

Importante: El uso indebido del sistema puede resultar en acciones disciplinarias de acuerdo con el reglamento del SENA.

Ayuda - Comandos de Voz
Importante: El navegador necesita tu permiso para acceder al micrófono.
🎤 Cómo Usar Comandos de Voz:
Activar el Micrófono
  1. Haz clic en el botón en la barra superior
  2. El navegador pedirá permiso para usar el micrófono
  3. Haz clic en "Permitir"
  4. El botón cambiará a indicando que está escuchando
Comandos Disponibles
Di esto: Acción:
"ir a equipos" Navega a la página de equipos
"ir a inventario" Navega a la página de inventario
"ir a reservas" Navega a la página de reservas
"ir a dashboard" o "inicio" Vuelve al dashboard
"ayuda" Muestra comandos disponibles
¡Listo para usar! Al iniciar sesión, automáticamente tienes acceso a todos los comandos de voz (buscar, crear, editar, eliminar).

Chrome / Edge:
  1. Haz clic en el icono 🔒 en la barra de direcciones
  2. Busca "Micrófono" en permisos
  3. Cambia a "Permitir"
  4. Recarga la página (F5)
Firefox:
  1. Haz clic en el icono junto a la URL
  2. Ve a "Permisos"
  3. Marca "Permitir" en micrófono
