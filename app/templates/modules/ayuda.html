Preguntas Frecuentes

Para iniciar sesión:

  1. Ingresa a la página principal del sistema
  2. Introduce tu ID de usuario (Ej: INST001)
  3. Haz clic en "Ingresar" o presiona Enter
  4. Si tienes reconocimiento facial habilitado, permite el acceso a la cámara
Nota: Si olvidaste tu contraseña, usa la opción "Recuperar Contraseña"

Para reservar un equipo:

  1. Ve al módulo de "Equipos" desde el menú lateral
  2. Busca el equipo que necesitas
  3. Haz clic en "Reservar"
  4. Selecciona la fecha y hora de inicio y fin
  5. Confirma la reserva

Puedes ver tus reservas activas en el módulo "Reservas".

Para usar comandos de voz:

  1. Haz clic en el ícono del micrófono en la barra superior
  2. Permite el acceso al micrófono cuando se solicite
  3. Di tu comando claramente (Ej: "Mostrar equipos disponibles")
  4. El sistema procesará y ejecutará tu comando
Comandos disponibles:
  • "Mostrar equipos"
  • "Ver inventario"
  • "Mis reservas"
  • "Estado del laboratorio"

Para configurar la accesibilidad:

  1. Haz clic en el ícono ♿ en la barra superior
  2. Activa las opciones que necesites:
    • Alto Contraste
    • Tamaño de Texto (ajustable)
    • Modo Oscuro
    • Reducir Animaciones
  3. Haz clic en "Guardar Configuración"

Tus preferencias se guardarán automáticamente.

Si encuentras un error:

  1. Anota el mensaje de error que aparece
  2. Toma una captura de pantalla si es posible
  3. Contacta al administrador del sistema
  4. Proporciona la siguiente información:
    • Tu ID de usuario
    • Fecha y hora del error
    • Qué estabas haciendo cuando ocurrió
    • Captura de pantalla

Bienvenido al Sistema de Gestión de Laboratorios

Este sistema te permite:

  • ✅ Consultar equipos y materiales disponibles
  • ✅ Reservar equipos para tus prácticas
  • ✅ Gestionar tu perfil de usuario
  • ✅ Acceder a reportes y estadísticas

  • Dashboard: Vista general con estadísticas en tiempo real

📅 Manual de Reservas

Cómo Reservar un Equipo
  1. Accede al módulo Equipos
  2. Busca el equipo que necesitas
  3. Verifica su disponibilidad
  4. Haz clic en "Reservar"
  5. Selecciona fecha y hora de inicio
  6. Selecciona fecha y hora de fin
  7. Confirma la reserva
Estados de Reserva
  • Programada Reserva confirmada y pendiente
  • Activa En uso actualmente
  • Completada Finalizada correctamente
  • Cancelada Anulada por el usuario
Importante: Cancela tus reservas con al menos 2 horas de anticipación para permitir que otros usuarios puedan usar el equipo.
Tip: Puedes ver todas tus reservas activas en el módulo "Reservas" del menú lateral.

♿ Guía de Accesibilidad

El sistema incluye múltiples opciones de accesibilidad para garantizar que todos los usuarios puedan utilizarlo cómodamente.

Alto Contraste

Aumenta el contraste visual entre texto y fondo para personas con baja visión.

Cómo activar: Haz clic en el botón ♿ en la barra superior y activa el switch "Alto Contraste"

Tamaño de Texto Ajustable

Ajusta el tamaño del texto desde 80% hasta 200% según tu preferencia.

Cómo usar: Usa el control deslizante en las opciones de accesibilidad

Modo Oscuro

Reduce el brillo de la pantalla con una paleta de colores oscuros.

Ideal para: Uso nocturno o personas con sensibilidad a la luz

Navegación por Teclado

Navega completamente sin mouse usando Tab, Enter y las flechas.

Compatible con: Lectores de pantalla NVDA y JAWS

🔐 Registro Facial

Cómo registrar tu rostro:

  1. Ve al módulo "Registro Facial"
  2. Haz clic en "Iniciar Cámara"
  3. Permite el acceso a la cámara
  4. Posiciona tu rostro en el centro
  5. Haz clic en "Capturar Rostro"
  6. Verifica la imagen capturada
  7. Haz clic en "Registrar"
Consejos:
  • Asegúrate de tener buena iluminación
  • Mira directamente a la cámara
  • Evita usar gafas oscuras o gorras
🎤 Comandos de Voz

Comandos disponibles:

  • "Mostrar equipos" - Muestra el catálogo de equipos
  • "Ver inventario" - Abre el módulo de inventario
  • "Mis reservas" - Muestra tus reservas activas
  • "Estado del laboratorio" - Información general
  • "Ir al dashboard" - Vuelve a la página principal
🔑 Autenticación JWT (API)

Para usar funciones de API:

  1. Haz clic en el botón 🔑 en la barra superior
  2. Ingresa tu ID de usuario
  3. Haz clic en "Iniciar"
  4. El token se guardará automáticamente

El token JWT permite crear y editar equipos desde la interfaz web.

🔓 ¿Olvidaste tu contraseña?

Proceso de recuperación:

  1. En la página de login, haz clic en "Recuperar Contraseña"
  2. Ingresa tu ID de usuario
  3. Recibirás un código de 6 dígitos por correo electrónico
  4. Ingresa el código en la página de verificación
  5. Establece tu nueva contraseña
  6. Confirma la nueva contraseña
Importante: El código de recuperación expira en 15 minutos.
Seguridad: Si no solicitaste el código, ignora el correo y contacta al administrador.

⚠️ Problemas Frecuentes
Problema: No puedo crear o editar equipos

Solución:

  1. Verifica que tengas permisos de Administrador (Nivel 4)
  2. Inicia sesión con el token JWT (botón 🔑)
  3. Si el error persiste, cierra sesión y vuelve a entrar
Problema: La cámara no funciona

Solución:

  1. Verifica que tu navegador tenga permisos para acceder a la cámara
  2. Cierra otras aplicaciones que puedan estar usando la cámara
  3. Prueba con otro navegador (Chrome o Edge recomendados)
  4. Verifica que tu cámara esté conectada correctamente
Problema: No recibo el código de recuperación

Solución:

  1. Verifica tu carpeta de spam/correo no deseado
  2. Asegúrate de haber ingresado tu ID correctamente
  3. Espera unos minutos, el correo puede tardar
  4. Si no llega, contacta al administrador
Problema: El sistema está lento

Solución:

  1. Limpia la caché de tu navegador
  2. Cierra pestañas innecesarias
  3. Verifica tu conexión a internet
  4. Recarga la página (F5)
